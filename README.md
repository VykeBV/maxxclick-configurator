# Maxxclick 3D Configurator

Drag-and-drop 3D product configurator for the Batavia Maxxclick rail storage
system. Built with Three.js, no build step — just static files.

## Run locally

```
python3 -m http.server 5173
open http://localhost:5173
```

## Embed

```html
<iframe src="https://<your-user>.github.io/maxxclick-configurator/"
  style="width:100%;height:720px;border:0;"
  loading="lazy"></iframe>
```
